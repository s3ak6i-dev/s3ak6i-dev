## Featured Project

### [Datrix](https://datrix-test.vercel.app) &nbsp;—&nbsp; AI Data Infrastructure Platform

The intelligence layer beneath every AI system. Datrix scans, cleans, enriches, and continuously improves the data that drives ML models.

**What it does:**
- Quality Engine — deep statistical analysis with quantified model impact per issue
- Pipeline Builder — generate full data pipelines from plain English in under 2 minutes
- Synthetic Data Engine — fill class imbalances with statistically validated synthetic examples
- Active Learning Loop — identify which unlabeled examples move your model most
- Compliance Autopilot — continuous GDPR, HIPAA, SOC 2 monitoring with auto-generated audit reports

**Stack:** `FastAPI` `React` `TypeScript` `PostgreSQL` `Cloudflare R2` `Three.js` `Docker` `Railway` `Vercel`

[Live App](https://datrix-test.vercel.app) &nbsp;·&nbsp; [API Docs](https://datrix-production-011a.up.railway.app/docs) &nbsp;·&nbsp; [Repo](https://github.com/s3ak6i-dev/Datrix)
